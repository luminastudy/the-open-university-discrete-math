# Discrete Mathematics

A Lumina Study course covering the fundamentals of discrete mathematics.

## Topics

- **Introduction to Logic** - Foundational concepts in mathematical logic
- **Set Theory** - Study of sets and their properties
- **Combinatorics** - Counting principles and techniques
- **Graph Theory** - Study of graphs and their applications

## Structure

```
Introduction to Logic
        ↓
    Set Theory
      ↓    ↓
Combinatorics  Graph Theory
```

## Usage

This course follows the [Lumina Study block schema](https://www.npmjs.com/package/@lumina-study/block-schema) v0.2.
